# bookkeeping

Double-entry bookkeeping toolkit for small businesses

## Quick Start

```bash
# Show info
./scripts/ledger-engine.sh --about

# Primary report
./scripts/ledger-engine.sh --journal

# Secondary report
./scripts/ledger-engine.sh --trial-balance
```
