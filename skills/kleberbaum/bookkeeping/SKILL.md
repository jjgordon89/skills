# bookkeeping

Manage journals, ledgers, and trial balances with double-entry bookkeeping logic.

## Journal Entries

Use `--journal` to access the journal entries functionality.

## Ledger Accounts

Use `--trial-balance` for the ledger accounts view.

## Trial Balance

Run `scripts/ledger-engine.sh --about` to display metadata and copyright.
