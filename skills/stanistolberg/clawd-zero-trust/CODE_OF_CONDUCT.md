# Code of Conduct

## The short version

Be professional. Be direct. Focus on the code.

This is a technical project. Discussions should be about security, implementation quality, and correctness. Personal attacks, harassment, or off-topic noise will result in removal from the project.

- Provide technical criticism with specifics, not opinions

## Unacceptable behavior

- Personal attacks or insults
- Publishing others' private information
- Harassment of any kind
- Deliberate introduction of vulnerabilities under the guise of contribution

## Enforcement

Violations reported to stan@blocksoft.tech. Maintainer decisions are final.
