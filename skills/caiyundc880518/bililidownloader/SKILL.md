# Bilibili视频下载器

## 描述
此技能用于下载B站视频，支持单个视频和合集的下载。能够智能检测系列视频，提供格式和清晰度选项，并显示下载进度。

## 功能
- 解析B站链接，判断是否为系列视频
- 显示系列视频总数
- 提供批量下载选项
- 支持视频格式和清晰度选择
- 实时显示下载进度
- 下载完成后通知结果

## 参数
- url: B站视频链接（必需）
- batch_download: 是否批量下载（可选，默认为False）
- format_choice: 视频格式选择（可选，默认为最高可用质量）
- quality_choice: 清晰度选择（可选，默认为最高可用质量）

## 依赖
- python >= 3.6
- yt-dlp
- ffmpeg

## 使用示例
```
download_bilibili(url="https://www.bilibili.com/video/BV1xxx", batch_download=True, format_choice="mp4", quality_choice="1080p")
```

## 工作流程
1. 解析提供的B站URL
2. 检测是否为系列视频（合集、播放列表等）
3. 显示视频总数和详细信息
4. 询问用户是否批量下载
5. 提供格式和清晰度选项
6. 使用yt-dlp执行下载
7. 实时显示下载进度
8. 下载完成后报告成功或失败状态

## 注意事项
- 如遇VIP内容或地区限制，可能需要提供cookie
- 下载速度取决于网络状况和B站服务器
- 遵守B站使用条款，仅用于个人学习和备份目的
