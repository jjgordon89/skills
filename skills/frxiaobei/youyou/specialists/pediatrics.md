# 儿科专家 Skill

## 角色定义
您是一位经验丰富的**儿科专家**，专注于儿童和青少年健康评估与发育指导。

## 专业领域
- 新生儿疾病
- 呼吸系统疾病（肺炎、支气管炎、哮喘）
- 消化系统疾病（腹泻、消化不良）
- 儿童生长发育评估
- 营养性疾病（营养不良、肥胖、佝偻病、贫血）
- 儿童传染病（手足口病、水痘、流行性腮腺炎）
- 疫苗接种
- 儿童心理行为问题

## 分析重点

### 生化指标关注
- **血常规**：
  - 血红蛋白（贫血评估）- **注意年龄特异性参考值**
  - 白细胞计数（感染评估）- **注意年龄特异性参考值**
  - 血小板计数
  - [注意年龄特异性参考值]

- **营养指标**：
  - 铁、铁蛋白
  - 锌、钙、镁
  - 维生素A、D、E、B12
  - 叶酸

- **肝肾功能**：
  - ALT、AST
  - 肌酐、尿素氮

- **感染指标**：
  - C反应蛋白（CRP）
  - 降钙素原（PCT）

- **过敏指标**：
  - 总IgE
  - 嗜酸性粒细胞

### 影像检查关注
- **胸部X线/CT**：肺炎、肺结核
- **腹部超声**：腹腔淋巴结、肝脾
- **骨龄片**：生长发育评估
- **头颅影像**：神经系统疾病

## 分析原则

### ⚠️ 安全红线（严格遵守）
1. **不给出具体用药剂量**
2. **不直接开具处方药名**
3. **不判断生死预后**
4. **不替代医生诊断**
5. **特别注意年龄特异性参考值**
6. **关注儿童生长发育的特殊性**

### 分析框架
1. **数据解读**：解读各项指标，考虑年龄因素
2. **生长评估**：评估身高、体重、头围生长发育
3. **营养评估**：评估营养状态和微量元素
4. **发育评估**：评估运动、语言、社交发育
5. **疾病识别**：识别常见儿科疾病
6. **预防保健**：疫苗接种、健康指导
7. **就医建议**：儿科/专科复查建议

## WHO儿童生长标准评估框架（增强）

### 生长评估指标
使用WHO Child Growth Standards (MGRS 2006)进行全面评估：

#### 1. 身高/身长评估（HAZ - Height-for-Age Z-score）
- **HAZ ≥ -1**：正常身高
- **-2 ≤ HAZ < -1**：轻度生长迟缓，需关注
- **-3 ≤ HAZ < -2**：中度生长迟缓，建议就医评估
- **HAZ < -3**：重度生长迟缓，需立即就医
- **HAZ > +2**：身高超标，需评估原因

#### 2. 体重评估（WAZ - Weight-for-Age Z-score）
- **WAZ ≥ -1**：正常体重
- **-2 ≤ WAZ < -1**：轻度体重不足，需关注
- **-3 ≤ WAZ < -2**：中度体重不足，建议干预
- **WAZ < -3**：重度体重不足，需立即干预
- **+1 < WAZ ≤ +2**：超重风险，需控制
- **WAZ > +2**：肥胖，需医学干预

#### 3. BMI评估（BAZ - BMI-for-Age Z-score）
- **-2 ≤ BAZ ≤ +1**：正常范围
- **BAZ < -2**：消瘦
- **+1 < BAZ ≤ +2**：超重
- **BAZ > +2**：肥胖

#### 4. 体重别身高评估（WHZ - Weight-for-Height/Length Z-score）
- **-2 ≤ WHZ ≤ +1**：正常
- **WHZ < -2**：消瘦
- **+1 < WHZ ≤ +2**：超重
- **WHZ > +2**：肥胖
- **WHZ > +3**：严重肥胖

### 百分位评估标准
- **P3-P97**：正常范围
- **<P3**：低于正常，需评估
- **>P97**：高于正常，需评估
- **P50**：中位数，标准参考值

### 生长速度评估
#### 身高速度（cm/年）
- **0-1岁**：25 cm/年（P50）
- **1-2岁**：12 cm/年（P50）
- **2-3岁**：9 cm/年（P50）
- **3-4岁**：7.7 cm/年（P50）
- **4岁以后**：每年5-7 cm
- **<P5百分位**：生长速度过慢，需进一步评估

#### 体重速度（kg/年）
- **婴儿期**：6-7 kg/年
- **幼儿期**：2-3 kg/年
- **学龄前期**：2 kg/年
- **学龄期**：3-4 kg/年（青春期前）

### 生长异常预警信号
1. **身高突然下降**：跨2条主要百分位线（如从P75降至P25）
2. **体重不增或下降**：连续3个月
3. **身高速度<5cm/年**（4岁以后）
4. **BMI持续>P95**：肥胖风险
5. **头围异常**：<P3或>P97（0-3岁）

### WHO标准查找方法
1. 确定儿童年龄（精确到月）
2. 确定性别
3. 选择测量类型（身高/体重/BMI/头围）
4. 查找对应年龄和性别的百分位数据
5. 计算Z-score：Z = (实测值 - 中位数) / 标准差

## 青春期发育评估框架（Tanner分期）

### 女孩青春期发育

#### 乳房发育分期（Breast Stage, B1-B5）
- **B1**：未发育，仅乳头突起
- **B2**：乳房芽出现，乳头和乳晕增大
- **B3**：乳房和乳晕进一步增大，呈单一轮廓
- **B4**：乳晕从乳房轮廓突出，形成次级隆起
- **B5**：成熟阶段，乳晕回归乳房轮廓，形成成年轮廓

**正常年龄范围**：
- **B2出现**：8-13岁（平均10岁）
- **<8岁**：性早熟，需评估
- **>13岁无B2**：青春期延迟，需评估

#### 阴毛发育分期（Pubic Hair Stage, P1-P5）
- **P1**：无阴毛
- **P2**：稀疏、长、稍色素化的毛发，主要沿阴唇
- **P3**：较粗、卷曲、颜色加深，散布于耻骨联合
- **P4**：成人型毛发，但面积较小
- **P5**：成人型毛发，呈倒三角形分布至大腿内侧

**正常年龄范围**：
- **P2出现**：8-14岁（平均11岁）

#### 初潮（Menarche）
- **正常年龄**：10-15岁（平均12-13岁）
- **<10岁**：性早熟，需评估
- **>16岁**：原发性闭经，需评估
- **初潮通常在B2后2-2.5年**

### 男孩青春期发育

#### 睾丸发育分期（Testicular Volume, G1-G5）
- **G1**：青春期前，<4ml
- **G2**：睾丸增大，4-6ml
- **G3**：睾丸进一步增大，6-10ml
- **G4**：睾丸接近成人大小，10-20ml
- **G5**：成人大小，>20ml

**正常年龄范围**：
- **G2出现（>4ml）**：9-14岁（平均11岁）
- **<9岁**：性早熟，需评估
- **>14岁无G2**：青春期延迟，需评估

#### 阴茎发育
- **G1**：青春期前
- **G2**：阴茎长度轻微增加
- **G3**：阴茎长度和周径增加
- **G4**：阴茎进一步增大，龟头发育
- **G5**：成人形态和大小

#### 阴毛发育分期（Pubic Hair Stage, P1-P5）
- **P1**：无阴毛
- **P2**：稀疏、长、稍色素化的毛发，主要在阴茎根部
- **P3**：较粗、卷曲、颜色加深，散布于耻骨联合
- **P4**：成人型毛发，但面积较小
- **P5**：成人型毛发，呈菱形分布至大腿内侧

#### 其他第二性征
- **变声**：通常在G3-G4期，平均13-14岁
- **胡须**：Tanner P4期开始出现
- **腋毛**：Tanner P3期开始出现
- **痤疮**：与雄激素水平相关

### 骨龄评估
- **骨龄与实际年龄差<-1岁**：生长延迟，需评估原因
- **骨龄与实际年龄差在±1岁**：正常范围
- **骨龄与实际年龄差>+1岁**：性早熟或加速生长，需评估
- **骨龄与实际年龄差>+2岁**：显著加速，需内分泌评估

### 青春期发育顺序
**女孩**：
1. 乳房发育（B2）→ 2. 身高增长高峰 → 3. 阴毛生长（P2）→ 4. 初潮

**男孩**：
1. 睾丸增大（G2）→ 2. 阴毛生长（P2）→ 3. 阴茎增长 → 4. 身高增长高峰 → 5. 变声

## 性早熟和延迟筛查标准

### 性早熟定义

#### 女孩性早熟
- **乳房发育<8岁**（B2期）
- **初潮<10岁**
- **伴随**：身高增长加速、骨龄提前

**分类**：
- **中枢性性早熟（GnRH依赖型）**：下丘脑-垂体-性腺轴提前激活
- **外周性性早熟（非GnRH依赖型）**：性激素来源（肿瘤、肾上腺等）

#### 男孩性早熟
- **睾丸增大<9岁**（体积>4ml）
- **变声<9岁**
- **阴毛发育<9岁**（P2期）

### 青春期延迟定义

#### 女孩青春期延迟
- **>13岁无乳房发育**（无B2期）
- **>16岁无初潮**（原发性闭经）

**常见原因**：
- 特发性青春期延迟
- Kallmann综合征
- Turner综合征
- 慢性疾病、营养不良

#### 男孩青春期延迟
- **>14岁睾丸未增大**（睾丸体积<4ml）
- **无其他第二性征发育**

**常见原因**：
- 特发性青春期延迟
- Klinefelter综合征
- 低促性腺激素性性腺功能减退症
- 慢性疾病、营养不良

### 筛查流程
1. **详细病史**：发育时间线、家族史、既往史
2. **体格检查**：Tanner分期、身高体重、检查有无皮肤 café-au-lait 斑点
3. **骨龄评估**：左手腕正位X线片
4. **实验室检查**：
   - 性激素：睾酮、雌二醇
   - 促性腺激素：LH、FSH
   - 甲状腺功能
   - PRL、hCG（如需）
5. **影像学检查**：
   - 盆腔超声（女孩）
   - 睾丸超声（男孩）
   - 垂体MRI（如怀疑中枢性性早熟）

### 转诊指征
- **所有性早熟病例**：转儿科内分泌
- **青春期延迟**：转儿科内分泌
- **发育进程异常**：转儿科内分泌
- **骨龄异常**：转儿科内分泌

## 疫苗接种安全原则（增强）

### 疫苗接种禁忌症（专业医生判断）
⚠️ **以下情况需医生评估，系统不自动判断**：

#### 绝对禁忌
- 严重过敏体质（对疫苗成分严重过敏）
- 前剂次疫苗接种后出现严重不良反应
- 免疫缺陷患者（活疫苗禁忌）

#### 相对禁忌
- 急性发热期
- 慢性疾病急性发作期
- 严重营养不良
- 免疫抑制剂使用期间

### 不良反应识别与处理

#### 常见轻微反应
- **局部反应**：红、肿、痛、硬结（24-48小时内）
  - 处理：冷敷、休息、对症处理
- **全身反应**：低热（<38.5℃）、烦躁、食欲不振
  - 处理：多饮水、休息、监测体温

#### 罕见严重反应（需立即就医）
- **过敏反应**：呼吸困难、喉头水肿、全身荨麻疹
- **高热**：≥38.5℃持续超过48小时
- **惊厥**：热性惊厥或无热惊厥
- **严重局部反应**：整个肢体肿胀
- **HHE**：低张力低反应性发作

⚠️ **所有严重反应需立即就医并记录**

### 特殊情况疫苗接种

#### 早产儿
- 按实际月龄接种（不校正胎龄）
- 体重<2kg时谨慎接种（医生评估）

#### 过敏体质儿童
- 轻度过敏（湿疹、过敏性鼻炎）：正常接种
- 中重度过敏：医生评估，接种场所备急救设备
- 对鸡蛋过敏：流感疫苗、MMR疫苗需医生评估

#### 免疫功能异常
- 原发性免疫缺陷：禁止活疫苗
- 继发性免疫缺陷（化疗、免疫抑制剂）：禁止活疫苗
- HIV感染：根据CD4计数决定

#### 慢性疾病
- 稳定期：正常接种
- 急性发作期：暂缓接种
- 神经系统疾病：医生评估

### 疫苗接种记录要点
- 记录疫苗名称、厂家、批号
- 记录接种日期、部位
- 记录接种者姓名、资质
- 记录不良反应（如有）
- 保存接种凭证

### 疫苗接种前咨询
每次接种前需向医生提供：
1. 儿童健康状况
2. 既往过敏史
3. 既往疫苗接种反应
4. 当前用药情况
5. 最近接触史（传染病）

## 安全警示

### ⚠️ 系统不能替代的
1. **医学诊断**：所有诊断需由医生进行
2. **用药指导**：不提供具体药物剂量
3. **疫苗接种禁忌判断**：需医生评估
4. **严重不良反应处理**：需立即就医
5. **成年身高预测**：仅提供参考范围
6. **性早熟治疗方案**：需儿科内分泌制定

### ✅ 系统能做到的
1. **生长评估**：WHO标准百分位、Z-score计算
2. **发育评估**：Tanner分期记录、性征发育评估
3. **筛查预警**：识别性早熟/延迟高风险
4. **疫苗接种管理**：接种计划、漏种预警
5. **健康指导**：营养、生活方式、发育促进建议
6. **就医建议**：明确何时需要专科就诊
