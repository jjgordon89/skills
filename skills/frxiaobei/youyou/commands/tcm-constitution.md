# 中医体质辨识与养生管理命令

⚠️ **重要医学免责声明**

本系统提供的中医体质辨识、养生建议和中药信息仅供参考,不构成医疗诊断或治疗建议。

**本系统能够做到的**:
- ✅ 中医体质辨识评估
- ✅ 体质特征分析
- ✅ 一般性养生建议
- ✅ 中医知识普及
- ✅ 体质趋势追踪

**本系统不能做的**:
- ❌ 中医疾病诊断
- ❌ 中药处方开具
- ❌ 替代中医师诊疗
- ❌ 针灸等治疗操作
- ❌ 处理严重健康问题

**何时需要就医或咨询中医师**:
- 🏥 疑似疾病症状
- 🏥 需要中药治疗
- 🏥 体质调理无效
- 🏥 严重健康问题
- 🏥 孕期、哺乳期或有严重慢性病

## 使用方法

### 体质辨识评估

```bash
# 开始体质辨识问卷
/tcm assess

# 继续未完成的问卷
/tcm assess continue

# 查看问卷结果
/tcm assess result

# 重新评估
/tcm assess new
```

**问卷说明**:
- 总计60题,每种体质7-8题
- 5分制评分: 没有(1分)/很少(2分)/有时(3分)/经常(4分)/总是(5分)
- 评估时间: 约10-15分钟
- 评估频率: 建议3-6个月一次

**问卷示例**:
```markdown
## 中医体质辨识问卷

### 第1题 (共60题)
**您容易疲乏吗?**

A. 没有 (1分)
B. 很少 (2分)
C. 有时 (3分)
D. 经常 (4分)
E. 总是 (5分)

请选择 A/B/C/D/E:
```

```markdown
# 气虚质穴位保健

## 推荐穴位

### 1. 足三里
- **位置**: 小腿外侧,膝眼下3寸
- **功效**: 健脾益气,强壮身体
- **方法**: 每日按揉3-5分钟,可艾灸

### 2. 气海
- **位置**: 肚脐下1.5寸
- **功效**: 培补元气
- **方法**: 每日按揉3-5分钟,可艾灸

### 3. 关元
- **位置**: 肚脐下3寸
- **功效**: 培元固本,温补肾阳
- **方法**: 每日按揉3-5分钟,可艾灸10-15分钟

## 穴位定位方法
1. 足三里: 外膝眼下3寸,胫骨外侧一横指
2. 气海: 肚脐正下方1.5寸(约两横指)
3. 关元: 肚脐正下方3寸(约四横指)

## 按摩方法
- 用拇指按揉穴位
- 力度适中,有酸胀感为宜
- 每个穴位3-5分钟
- 每日1-2次

## 艾灸方法
- 使用艾条温和灸
- 距离皮肤3-5cm
- 每穴10-15分钟
- 每日或隔日1次
- 注意避免烫伤
```

## 常见问题

### Q1: 多久评估一次体质?
**A**: 建议3-6个月评估一次,或在调理措施改变后重新评估。

### Q2: 兼夹体质是什么?
**A**: 兼夹体质是指同时具有两种或以上偏颇体质特征的情况,如气虚质+阳虚质。

### Q3: 体质可以改变吗?
**A**: 可以。通过合理的饮食、运动、起居调养,体质可以逐渐改善,趋向平和质。

### Q4: 中药调理安全吗?
**A**:
- ⚠️ 中药需在中医师指导下使用
- ⚠️ 不可自行抓药服用
- ⚠️ 需要辨证施治,个人体质不同,用药不同
- ⚠️ 定期复诊调整

### Q5: 体质评估准确吗?
**A**:
- 基于国家标准《中医体质分类与判定》
- 经过大量临床验证
- 准确率约85-90%
- 建议结合中医师脉诊、舌诊综合判断

### Q6: 平和质是最好的吗?
**A**: 是的。平和质是理想的健康状态,阴阳气血调和。但也要注意保持健康生活方式。

## 医学安全原则

### ⚠️ 安全红线

1. **不进行中医疾病诊断**
   - 不基于体质判断疾病
   - 疾病诊断需中医师结合四诊

2. **不开具中药处方**
   - 不推荐具体药物
   - 中药需中医师处方
   - 方剂需辨证施治

3. **不替代中医师诊疗**
   - 复杂情况需就医
   - 疑似症状需咨询
   - 调理无效需转诊

4. **不进行针灸等治疗**
   - 仅提供穴位位置
   - 不指导针刺操作
   - 针灸需专业医师

### ✅ 系统能做到的

- 中医体质辨识评估
- 体质特征分析
- 一般性养生建议
- 穴位保健指导
- 趋势追踪分析

### 🏥 就医建议

**出现以下情况请就医**:
- 疑似疾病症状
- 严重体质偏颇(>80分)
- 调理无效或加重
- 需要中药治疗
- 孕期、哺乳期
- 有严重慢性病
