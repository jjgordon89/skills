# 医院就诊准备指南

在去医院前快速获取就诊准备信息，包括科室推荐、所需证件、注意事项和健康数据摘要。

## 操作说明

**参数说明：**
- `target`（可选）：可以是以下任一类型
  - **症状描述**：如"头疼"、"胃疼"、"咳嗽"等
  - **科室名称**：如"心内科"、"消化科"等
  - **检查项目**：如"体检"、"B超"、"CT"等
  - **不填**：显示通用就诊准备指南和健康摘要

**示例：**
```
/prepare              # 显示通用准备指南和健康摘要
/prepare 头疼         # 根据症状推荐科室
/prepare 心内科        # 直接获取科室指南
/prepare 体检         # 检查项目准备
/prepare 胸闷气短     # 多症状描述
```

## 默认行为（无参数）

当用户不输入任何参数时（`/prepare`），执行以下操作：

### 1. 读取用户健康数据

从系统中读取：
- 最近的症状记录（最近7天）
- 正在使用的药物
- 最近的检查结果（最近30天）
- 最近的诊断
- 待复查项目

### 2. 分析就诊需求

根据最近的症状和数据，智能判断：
- 是否建议就医
- 可能需要去的科室
- 紧急程度评估

### 3. 显示通用准备指南

包括：
- 必备证件清单
- 资料准备清单
- 就诊流程说明
- 就诊技巧

### 4. 输出健康摘要

展示用户当前的健康状况摘要，方便就诊时向医生展示

## 执行步骤

### 1. 智能识别就诊目标

从用户输入中识别：
- **如果是症状** → 推荐科室
- **如果是科室** → 直接提供该科室准备指南
- **如果是检查** → 提供检查准备指南

| 症状关键词 | 推荐科室 | 备注 |
|----------|---------|------|
| 头疼、头晕、眩晕 | 神经内科 | 如伴高血压可挂心内科 |
| 胸痛、胸闷、心慌 | 心内科 | 紧急情况挂急诊 |
| 咳嗽、咳痰、呼吸困难 | 呼吸内科 | |
| 胃疼、肚子疼、腹泻 | 消化内科 | 急性剧痛挂急诊 |
| 发烧、发热 | 发热门诊或内科 | |
| 皮疹、瘙痒 | 皮肤科 | |

### 3. 生成就诊准备清单

#### 必备证件清单

**通用证件：**
- ☐ 身份证/医保卡/社保卡（必带）
- ☐ 医保卡/就诊卡（必带）
- ☐ 银行卡或手机支付（备）
- ☐ 既往病历资料（如有）
- ☐ 检查报告和影像资料（如有）

**过敏相关：**
- ☐ 过敏史清单（必带）⭐
- ☐ 过敏急救药物（如严重过敏或过敏性休克）⭐

**特殊情况：**
- 住院：身份证、医保卡、住院押金
- 转诊：转诊单、上级医院病历
- 医保报销：相关发票和清单
- 异地就医：异地就医备案证明

#### 资料准备清单

**如果有以下资料，请携带：**
- ☐ 既往出院小结
- ☐ 最近的检查报告（血常规、生化等）
- ☐ 影像资料（CT、MRI、X光片及报告）
- ☐ 用药清单或正在服用的药物照片
- ☐ 既往手术记录
- ☐ 过敏史清单（必带）⭐
- ☐ 过敏急救药物（如严重过敏）⭐
- ☐ 家族病史信息

### 4. 就诊注意事项

#### 就诊前准备

**空腹要求：**
- 需要空腹的检查：空腹血糖、肝功能、血脂、腹部B超、胃肠镜等
- 空腹时间：至少8-12小时不进食
- 可以喝少量白开水（不超过200ml）

**穿着建议：**
- 穿宽松衣物，方便检查
- 不穿连体衣、连衣裙
- 不戴首饰（检查时需取下）
- 穿易于穿脱的鞋子

**其他准备：**
- 保证充足睡眠，避免过度疲劳
- 如测量血压，就诊前30分钟避免吸烟、饮咖啡、剧烈运动
- 女性患者：月经期告知医生（某些检查需避开月经期）
- 如有晕血、晕针史，提前告知

#### 就诊流程

```
1. 预约挂号
   ↓
2. 到达医院（建议提前15-30分钟）
   ↓
3. 自机取号或签到
   ↓
4. 候诊区等待叫号
   ↓
5. 医生问诊和检查
   ↓
6. 医生开检查单或处方
   ↓
7. 缴费（检查费/药费）
   ↓
8. 做检查/取药
   ↓
9. 等检查结果（如需要）
   ↓
10. 复诊给医生看结果
```

#### 就诊技巧

**如何高效沟通：**
1. **说清主要症状**：哪里不舒服、多久了、有什么特点
2. **讲明病史**：既往病史、手术史、**过敏史（非常重要！）**
3. **列出用药**：正在服用的药物名称和剂量
4. **提出疑问**：准备好想问医生的问题
5. **理解医嘱**：不清楚的及时询问，确保明白

**⚠️ 特别强调过敏史：**
- **第一时间主动告知**：就诊开始时立即告知过敏史
- **说明严重程度**：特别是严重过敏和过敏性休克史
- **具体过敏原**：明确说明对什么药物/食物过敏
- **过敏反应**：描述当时的过敏反应症状
- **药物家族**：如对青霉素过敏，应告知医生避免使用所有青霉素类药物
- **检查提醒**：如对碘造影剂过敏，做CT/MRI增强检查时要提前告知
- **携带急救药**：有过敏性休克史的患者，应随身携带肾上腺素笔等急救药物

**建议过敏史沟通话术：**
```
"医生，我有过敏史需要告知您：
- 对青霉素严重过敏，出现过呼吸困难，请避免使用青霉素类药物
- 对碘造影剂过敏，做增强CT时要特别注意
- 有蜜蜂叮咬过敏性休克史，已携带肾上腺素笔"
```

**建议提问清单：**
- 我得了什么病？
- 需要做哪些检查？
- 严重吗？需要治疗吗？
- 这个药怎么吃？吃多久？
- 有什么副作用吗？
- 饮食和生活需要注意什么？
- 什么时候需要复查？
- 出现什么情况需要立即就医？

### 6. 科室专项准备

#### 心内科

**常见症状：** 胸痛、胸闷、心慌、气短、水肿、高血压

**常见检查：** 心电图、心脏彩超、动态心电图、冠脉造影

**准备事项：**
- 检查当天穿宽松衣物
- 24小时动态心电图需提前预约
- 冠脉造影需空腹，并有人陪同

#### 消化内科

**常见症状：** 胃疼、肚子疼、反酸、恶心、呕吐、腹泻、便秘、黄疸

**常见检查：** 胃镜、肠镜、腹部B超、CT

**准备事项：**
- 胃镜：空腹6-8小时
- 肠镜：前一天流食，检查当天空腹，需服用泻药清肠
- 腹部B超：空腹8-12小时

#### 呼吸内科

**常见症状：** 咳嗽、咳痰、呼吸困难、胸痛、发热

**常见检查：** 胸片/胸部CT、肺功能、血气分析

**准备事项：**
- 胸部CT一般无需特殊准备
- 肺功能检查：检查前2小时禁食、禁烟

#### 内分泌科

**常见症状：** 糖尿病、甲状腺问题、骨质疏松、肥胖

**常见检查：** 血糖、糖化血红蛋白、甲状腺功能、骨密度

**准备事项：**
- 空腹血糖、糖化血红蛋白需空腹8-12小时
- 甲状腺功能一般无需空腹，但建议上午检查

#### 神经内科

**常见症状：** 头疼、头晕、失眠、抽搐、肢体麻木无力

**常见检查：** 头颅CT/MRI、脑电图、颈动脉彩超

**准备事项：**
- 头颅MRI需去除所有金属物品
- 脑电图需洗头，勿用发胶

#### 骨科

**常见症状：** 关节痛、腰痛、骨折、运动损伤

**常见检查：** X光、CT、MRI

**准备事项：**
- 带既往X光片、CT片（如有）
- 有内固定的需告知医生

#### 妇科

**常见症状：** 月经不调、腹痛、白带异常、不孕

**常见检查：** 妇科检查、B超、TCT、HPV

**准备事项：**
- 检查前3天避免性生活
- 检查前24-48小时不要阴道冲洗或用药
- 避开月经期（除非检查月经相关问题）

#### 泌尿外科

**常见症状：** 尿频、尿急、尿痛、血尿、排尿困难

**常见检查：** 尿常规、泌尿系B超、CT

**准备事项：**
- 泌尿系B超需憋尿
- 尿常规留取中段尿

### 8. 特殊检查准备指南

#### 体检

**准备事项：**
- 空腹10-12小时
- 前一天清淡饮食，不饮酒
- 保证充足睡眠
- 穿宽松衣物
- 女性避开月经期（妇检、尿检）

#### 血液检查

**空腹项目：** 空腹血糖、肝功能、肾功能、血脂、血流变

**准备：** 空腹8-12小时，可少量饮水

#### 尿常规

**准备：** 留取中段尿（清洁外阴后，先排一点尿，再留取中段尿液）

#### 大便常规

**准备：** 取蚕豆大小新鲜粪便，避免混入尿液或水

#### 胃镜

**准备：**
- 空腹6-8小时
- 检查前排空膀胱
- 如做无痛胃镜，需有人陪同

#### 肠镜

**准备：**
- 前一天流食
- 检查前服用泻药清肠
- 检查当天空腹
- 需有人陪同

#### B超（腹部）

**准备：**
- 肝胆胰脾：空腹8-12小时
- 泌尿系（肾、输尿管、膀胱）：需憋尿
- 妇科B超：需憋尿（如不做阴道B超）

#### CT

**准备：**
- 头部/胸部：一般无需特殊准备
- 腹部：空腹
- 增强CT：需空腹，并做碘过敏试验

#### MRI

**准备：**
- 去除所有金属物品（首饰、手机、钥匙、银行卡等）
- 体内有金属植入物（起搏器、金属夹等）需告知医生
- 幽闭恐惧症患者可能需要镇静

#### 心电图

**准备：**
- 检查前休息5-10分钟
- 保持情绪平稳
- 穿宽松上衣

### 9. 常见症状自检

在去医院前可以先进行简单自检：

**发热：**
- 测量体温
- 记录发热时间、最高体温
- 是否伴有寒战、出汗

**疼痛：**
- 疼痛部位（哪里疼）
- 疼痛性质（胀痛、刺痛、绞痛、隐痛）
- 疼痛程度（轻度、中度、重度）
- 持续时间（偶尔、持续几天、持续几周）
- 诱发因素（什么情况下加重或缓解）

**咳嗽：**
- 干咳还是有痰
- 痰的颜色（白痰、黄痰、血痰）
- 咳嗽时间（晨起、夜间、全天）

**腹泻/便秘：**
- 次数（每天几次）
- 形状（稀便、水样便、羊屎样）
- 是否伴有腹痛、恶心、呕吐
