# 性健康管理命令

## 医学免责声明

⚠️ **重要提示**:本系统仅用于健康追踪和教育目的,不提供医学诊断或治疗建议。

- 所有性健康问题应咨询专业医生或泌尿科/妇科医生
- 紧急情况应立即就医(如剧烈疼痛、异常出血等)
- 本系统不能替代专业医疗检查和诊断
- 请遵循医生的专业建议和治疗方案
- 性健康问题可能涉及心理因素,建议必要时咨询心理医生

## 功能概述

性健康管理模块提供全面的性健康追踪功能,包括:
- **男性性健康评估**:IIEF-5交互式问卷、性欲评估、射精功能
- **女性性健康评估**:FSFI评分、性欲评估、性交痛评估
- **STD筛查管理**:HIV、梅毒、淋病、衣原体等7项筛查
- **避孕管理**:7种避孕方法的有效性和副作用追踪
- **性活动日志**:频率、满意度、保护措施记录
- **用药管理**:PDE5抑制剂等性健康药物追踪
- **综合分析**:趋势分析、风险评估、个性化建议

### 2. IIEF-5 交互式问卷 - `iief5`

**国际勃起功能指数-5 (IIEF-5)** 是评估男性勃起功能的标准化问卷,包含5个问题,每个问题0-5分,总分0-25分。

#### 问卷问题

**问题1**:在过去6个月内,您有多少次对自己的勃起感到有信心?
- **评估重点**:勃起信心
- **0分**:无性活动
- **1分**:几乎没有/从不
- **2分**:少数几次(远少于一半时候)
- **3分**:有时(约一半时候)
- **4分**:多数时候(远多于一半时候)
- **5分**:几乎总是/总是

**问题2**:在过去6个月内,当您受到性刺激时,有多少次能够勃起?
- **评估重点**:勃起获得
- **0分**:无性活动
- **1分**:几乎没有/从不
- **2分**:少数几次(远少于一半时候)
- **3分**:有时(约一半时候)
- **4分**:多数时候(远多于一半时候)
- **5分**:几乎总是/总是

**问题3**:在过去6个月内,您有多少次能够插入伴侣?
- **评估重点**:插入能力
- **0分**:无性活动
- **1分**:几乎没有/从不
- **2分**:少数几次(远少于一半时候)
- **3分**:有时(约一半时候)
- **4分**:多数时候(远多于一半时候)
- **5分**:几乎总是/总是

**问题4**:在过去6个月内,您有多少次能够维持勃起完成性交?
- **评估重点**:勃起维持
- **0分**:无性活动
- **1分**:几乎没有/从不
- **2分**:少数几次(远少于一半时候)
- **3分**:有时(约一半时候)
- **4分**:多数时候(远多于一半时候)
- **5分**:几乎总是/总是

**问题5**:在过去6个月内,您尝试性交时有多少次感到满足?
- **评估重点**:性交满意度
- **0分**:无性活动
- **1分**:几乎没有/从不
- **2分**:少数几次(远少于一半时候)
- **3分**:有时(约一半时候)
- **4分**:多数时候(远多于一半时候)
- **5分**:几乎总是/总是

#### 评分标准

| 总分 | ED严重程度 | 建议 |
|------|-----------|------|
| 22-25 | 正常勃起功能 | 继续保持健康生活方式 |
| 17-21 | 轻度ED | 生活方式调整,如持续建议就医 |
| 12-16 | 轻中度ED | 建议咨询医生评估原因 |
| 8-11 | 中度ED | 建议就医,可能需要药物治疗 |
| 5-7 | 重度ED | 需要就医进行全面评估 |

#### 使用示例

```bash
/sexual iief5                                     # 启动交互式问卷
/sexual iief5 score 18                            # 直接记录总分
/sexual iief5 q1:4 q2:3 q3:4 q4:3 q5:4           # 记录各题得分
/sexual iief5 trend                               # 查看评分趋势
```

### 3. 女性性功能指数 - `fsfi`

**女性性功能指数 (FSFI)** 是评估女性性功能的标准化问卷。

#### FSFI 包含6个维度
1. **性欲** (Desire):2个问题
2. **性兴奋** (Arousal):4个问题
3. **阴道润滑** (Lubrication):4个问题
4. **性高潮** (Orgasm):3个问题
6. **满意度** (Satisfaction):3个问题
7. **疼痛** (Pain):3个问题

#### 使用示例

```bash
/sexual fsfi                                      # 启动FSFI问卷
/sexual fsfi score 28.5                           # 直接记录总分
/sexual fsfi desire 3.2 arousal 4.0              # 记录各维度得分
```

## 何时寻求专业帮助

### 男性警示信号

**立即就医(24小时内)**:
- 睾丸剧烈疼痛或扭转
- 异常持续勃起(>4小时)
- 尿道大量出血

**尽快就医(1周内)**:
- 勃起困难持续超过3个月
- 性欲显著下降影响生活质量
- 射精疼痛或异常
- 睾丸肿块或硬度变化
- 尿道异常分泌物

**定期检查**:
- 40岁以上:每年前列腺检查
- 50岁以上:每年PSA筛查
- 高风险:每3-6个月STD筛查

### 女性警示信号

**立即就医(24小时内)**:
- 剧烈盆腔疼痛
- 异常大量阴道出血
- 高热伴下腹痛

**尽快就医(1周内)**:
- 性交疼痛持续存在
- 性欲丧失影响关系
- 阴道异常出血或分泌物
- 外阴异常或肿块
- 复发性阴道炎

**定期检查**:
- 21-65岁:每3年巴氏涂片
- 30-65岁:每5年HPV+巴氏涂片
- 性活跃:每年盆腔检查
- 高风险:每3-6个月STD筛查

## 常见问题

**Q: IIEF-5问卷多久做一次?**
A: 建议每3-6个月评估一次,或在治疗/生活方式改变后重新评估。

**Q: STD筛查需要空腹吗?**
A: 一般不需要,但具体要求取决于检测类型和医疗机构。

**Q: 避孕套影响性快感怎么办?**
