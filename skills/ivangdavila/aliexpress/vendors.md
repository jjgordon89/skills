# Vendor Evaluation — Aliexpress

## Complete Vendor Check

Before recommending ANY purchase >$20, verify:

### Store Metrics
```
□ Store age: >2 years preferred
□ Followers: >5,000 minimum
□ Products: >50 (established catalog)
□ Positive feedback: >97%
□ Detailed seller rating: >4.7 all categories
```

### Communication Test
- Send pre-purchase question
- Response in <24h = good sign
- Generic copy-paste = caution
- Specific answers = trustworthy

### Review Analysis
| Review Type | Trust Level |
|-------------|-------------|
| Photo review, detailed text | High |
| Photo review, short text | Medium |
| Text only, specific | Medium |
| Text only, generic | Low |
| No reviews | Avoid |

Look for reviews mentioning:
- Actual product quality vs photos
- Real shipping time
- Packaging condition

## Red Flags

| Signal | Risk |
|--------|------|
| Store <6 months old | High scam risk |
| Only 5-star reviews | Fake reviews |
| Prices 50%+ below competitors | Bait and switch |
| Stock photos everywhere | Dropship middleman |
| No response to messages | Poor support |

## Verification Steps

1. **Image search**: Right-click product photo → Search image → Same product elsewhere cheaper?
2. **Price check**: Search product name on Aliexpress, sort by orders
3. **Factory check**: Multiple vendors with identical photos = same factory, buy from highest rated

## Trusted Store Indicators

- "Top Brand" badge (verify independently)
- "Aliexpress Choice" label
- Local warehouse option with tracking
- Multiple years anniversary badge
- High repeat buyer percentage
