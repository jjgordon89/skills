# Active Tasks

Track all background work here. Heartbeats should check this file first.

## Entry Format
```
### [TASK_ID] Short Description
- **Status:** RUNNING | COMPLETED | FAILED | STALE
- **Started:** ISO timestamp
- **Type:** subagent | background_exec | cron_check
- **Session/Process:** identifier
- **Expected Done:** timestamp or duration estimate
- **Check Cron:** cron job ID (if scheduled)
- **Report To:** how to notify user (telegram, session, etc.)
- **Result:** (filled on completion)
```
